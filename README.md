# Cloud Lab

Hands-on AWS and Terraform labs focused on cloud concepts.

## Purpose

This repository is used for studying, experiments, and infrastructure practice through hands-on labs.

The main goal is to deepen knowledge in:

* AWS cloud infrastructure
* Terraform and Infrastructure as Code (IaC)
* Networking fundamentals
* Security concepts
* Cloud architecture
* Infrastructure organization and best practices

Each lab focuses on one or more infrastructure concepts and services through practical hands-on implementations. Some labs are intentionally small and isolated, while others combine multiple concepts into more complete infrastructure scenarios.

## Labs

* **[basic-vpc](./basic-vpc)** — Foundational AWS networking lab with VPCs, subnets, route tables, Internet Gateway, and NAT Gateway.

* **[vpc-ec2-security](./vpc-ec2-security)** — AWS networking and security lab with EC2 instances, Security Groups, SSH access, and bastion-style architecture.
